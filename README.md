# 💰 Finance Dashboard

## 📌 Project Overview

This interactive Finance Dashboard was developed in **Power BI** to analyze financial performance through key business metrics. It provides insights into revenue, profit, costs, and product performance using dynamic visualizations and interactive filters.

---

## 🎯 Business Objectives

- Monitor overall financial performance
- Track revenue and profit trends
- Analyze Cost of Goods Sold (COGS)
- Identify top-performing products
- Compare country and segment performance
- Support data-driven financial decisions

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- CSV Dataset

---

## 📊 Dashboard Features

- Revenue KPI
- Profit KPI
- COGS Analysis
- Sales Trend Analysis
- Product Performance
- Country-wise Performance
- Segment Analysis
- Interactive Slicers

---

## 📈 Key Insights

- Revenue distribution by country
- Top-performing products
- Profitability trends
- Segment-wise contribution
- Financial performance comparison
